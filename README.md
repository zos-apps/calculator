## About

A simple calculator for basic math operations with a classic macOS design.

## Features

- Basic arithmetic
- Percentage calculations
- Memory functions
- Keyboard support

## Installation

### From App Store
Open the **App Store** in zOS and search for "Calculator".

### Via npm
```bash
npm install @zos-apps/calculator
```

## Usage

- Click buttons or use keyboard
- AC clears all
- C clears current entry

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `0-9` | Numbers |
| `+-*/` | Operations |
| `Enter/=` | Equals |
| `Esc` | Clear |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
